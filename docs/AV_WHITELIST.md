# 用户加白名单说明

本客户端使用 DLL 注入保护指标源码，**可能被杀毒软件误报**。请按以下步骤添加信任。

## Windows Defender / SmartScreen

1. 若提示「Windows 已保护你的电脑」，点 **更多信息** → **仍要运行**
2. 设置 → 隐私和安全性 → Windows 安全中心 → 病毒和威胁防护 → 管理设置 → **排除项** → 添加：
   - 客户端 `.exe` 所在文件夹
   - 文华 WT8 安装目录

## 360 安全卫士

1. 若拦截，点 **信任该程序** 或 **添加信任**
2. 木马查杀 → 信任区 → 添加文件/文件夹（客户端 exe + 文华目录）

## 火绒

1. 弹窗选择 **允许**
2. 防护中心 → 信任区 → 添加文件

## Win11 内存完整性（若注入失败）

设置 → 隐私和安全性 → Windows 安全中心 → 设备安全性 → 内核隔离 → 内存完整性 → 暂时关闭后重试（或保持白名单）

## 交付建议

- 用 **zip 压缩包** 发送，附带本说明
- 不要通过微信直接传裸 exe（易被云查杀）
